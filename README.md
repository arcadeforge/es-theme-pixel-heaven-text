# RetroPie Emulationstation Theme: es-pixel-heaven-text

This theme is made for 15khz CRTs based on 320x240. Use pi2scart or pi2jamma to connect your Rpi to a CRT.

* Installation
  * This is installable via RetroPie-Setup
